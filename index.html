Sistema básico de gerenciamento de vinhos

Sobre o sistema

Nesta pagina simples, é perguntado ao usuario os dados de seu vinho, e apos coletar os dados, são exibidos no console do navegador operante no momento.

Funcionalidades

  • Cadastro de vinhos
  • Entrada de dados via prompt()
  • Exibição com console.log()
  • Alertas informativos com alert()

Informações solicitadas

  • Nome do vinho
  • Tipo (Tinto, Branco ou Rosé)
  • Safra (ano)
  • Quantidade em estoque
